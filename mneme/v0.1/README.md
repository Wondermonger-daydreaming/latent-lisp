# Lisp+ — a research program, not (yet) a language

Founding bet: a language surface designed as **error-correcting code for
probabilistic readers** measurably reduces their structural and semantic
error. Reshaped by cross-lineage review into four planes (semantic /
epistemic / authority / temporal) where every design clause is a
preregistered hypothesis with a gate.

Rehydration ritual (in order): `HANDOFF.md` → `CONSTITUTION.md` →
`EXPERIMENTS.md` → `ledger/`.

If you have 4 hours: run E0. It can no-go the founding thesis for the
price of an afternoon, which is exactly what makes it worth running.
