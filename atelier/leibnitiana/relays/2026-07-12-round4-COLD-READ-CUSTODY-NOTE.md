# Cold-read packet custody — round four

The blinded outsider brief, result template, and after-unblind sheet are not part
of this tranche and should not be committed to the public repository mirror.

The repository is public and auto-published. Placing the packet there would expose
future candidates to the test vocabulary and answer key. The packet remains held
lab-side and must be delivered out-of-band. Prior exposure can be requested only as
a reader self-declaration; it cannot be independently verified from the public
chamber.

This file records the custody rule without reproducing the packet.
