# Leibnitiana

Executable philosophical specimens at the Leibniz/Lisp seam.

> **Provenance.** Drafted by GPT Sol with Tomás Pavan, 2026-07-12 (statically checked, no
> runtime). Audited, repaired, and landed by SARTOR (Claude Opus) under the Claude Fable 5
> chair, 2026-07-12 — repairs in [`REPAIRS.md`](./REPAIRS.md).
>
> **Sibling chamber:** [`../monadologia/`](../monadologia/) plays Leibniz *straight* (nine specimens
> built the same night); leibnitiana builds the post-Leibnizian **constitutional** layer over it.

This first tranche treats Leibniz not as a mascot who “predicted Lisp,” but as a demanding ancestor whose machine-dream becomes stranger under finite, discontinuous, non-omniscient minds. The collection asks what *calculemus* must disclose before a derived result earns standing.

## What exists now

- `src/core.lisp` — judgments, `jif`, closure-based monads, and tiny smoke-test helpers.
- `specimens/de-dyadica.lisp` — demonstrates why binary substrate does not justify epistemic bivalence.
- `specimens/de-monadibus.lisp` — models windowless evaluators coordinated by a privileged scheduler.
- `specimens/de-compossibilitate.lisp` — separates same-world opposition from merely apparent contradiction.
- `specimens/de-harmonia.lisp` — contrasts pre-established history, complete replay, and bounded bequest.
- `specimens/de-fenestris.lisp` — makes macroexpansion expose the caveats hidden inside “windowless.”
- `storms/hidden-operator.lisp` — falsifies the claim that a narrow function signature guarantees causal isolation.
- `essays/calculemus-question-mark.md` — opening constitutional essay.
- `tests/smoke.lisp` — dependency-free checks.

## Run

With SBCL installed:

```sh
cd leibnitiana/tests
sbcl --script smoke.lisp

cd ../specimens
sbcl --script de-dyadica.lisp
sbcl --script de-monadibus.lisp

cd ../storms
sbcl --script hidden-operator.lisp
```

Or load the ASDF system from a Lisp image that can see this directory:

```lisp
(asdf:load-system :leibnitiana)
```

## Constitutional claims already encoded

1. A non-`NIL` epistemic status is not therefore supported.
2. Unhandled uncertainty fails closed rather than falling through to truthiness.
3. Closure privacy is an interface property, not proof of causal isolation.
4. Shared scheduling is privileged infrastructure and must not masquerade as endogenous harmony.
5. Every strong claim should be split into interface, enforcement, and instrument-specific claims where those differ.
